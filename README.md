# Financial-SQL-Analytics-Project
This project demonstrates advanced SQL analytics on financial transaction data, including fraud pattern detection, customer lifetime value analysis, and merchant intelligence reporting.

## 🎯 Objectives
- Detect suspicious financial behavior using SQL
- Perform customer lifetime value analysis
- Analyze merchant category performance
- Build risk monitoring dashboards

## 🧰 Tech Stack
- PostgreSQL / MySQL
- SQL (CTEs, Window Functions, Aggregations)
- Power BI (Dashboard)

## 📊 Key Analyses
- Monthly transaction trends
- High-frequency fraud detection
- Spending spike anomalies
- Rolling average outlier detection
- Customer lifetime value (CLV)

## 🧠 Skills Demonstrated
- Complex joins
- Window functions
- Fraud detection logic
- Business analytics storytelling
